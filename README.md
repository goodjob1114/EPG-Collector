### EPG
 
電子節目表（Electronic Program Guide），通常伴隨數位電視訊號或數位廣播訊號傳送。
這些訊號可以透過有線電視、衛星電視、或地面電視被接收。
典型的接收裝置包括電視機及機頂盒，使用者可以看到當前節目及未來節目的更多的訊息。
EPG 通常在傳輸流中廣播，包括類似 MPH 的圖文訊息。

### EPG Collector

EPG 收集器，旨在收集各大電視臺的電子節目單，並儲存到資料庫。
